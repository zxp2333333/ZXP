Vim

Move In Vim

h move left

j move down

k move up

l move right

ctrl+e scroll the window down

ctrl+y scroll the window up

ctrl+f srcoll down one page

ctrl+b scroll up one page

H move cursor to the top of the window

M move cursor to the middle of the window

L move cursor to the bottom of the window

gg go to top of file

p>w move to the first letter of next word

b move to the first letter of previous word

e move to the last letter of next word

ge move to the last letter of previous word

G move to the last line of the file

t find until the next letter

; repeat the search operation again

, repeat the search operation reversely

Text Objection

w words

s sentences

p paragraph

t tags

a represent "around"

i represent "in"

f represent "find forward"

F represent "find backward"

g~~ change the case of the line

gUU change the line to the uppercase

guu change the line to the lowercase

gt go to the next file

ctrl+o go back to previous file

ctrl+0 go to side menu

ctrl+2 go to another page

space select the file in side menu

l move cursor when you select the file in side menu
